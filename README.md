# generative-art-studio
Where all of my generative art experiments live

## Setup

### Kotlin and OPENRNDR

- need git
- snap
  - sudo apt install -y snapd
- kotlin 
  - using `sdkman`
    - https://sdkman.io/
  - sdk install kotlin
  - skd install gradle


[Openrndr guide](https://guide.openrndr.org/#/02_Getting_Started_with_OPENRNDR/C00_SetupYourFirstProgram)
